# 📊 מערכת ניהול מס אישית

מערכת לניהול וחישוב מס הכנסה לישראל - מתאימה למי שיש לו מספר מקורות הכנסה בשנה אחת.

## ✨ יכולות

- 💼 **הכנסה כשכיר** - מעקב אחר משכורות ומס שנוכה
- 📋 **דמי אבטלה** - כולל מס שנוכה במקור
- 🎖️ **מילואים** - פטור ממס + נקודות זיכוי אוטומטיות
- 💰 **הכנסה כעצמאי** - מעקב הכנסות והוצאות מוכרות
- 🏛️ **מקדמות מס הכנסה** - מעקב חודשי ומצטבר
- 🏥 **ביטוח לאומי** - מעקב תשלומים נפרד
- ⭐ **נקודות זיכוי** - כולל גרוש משלם מזונות
- 🧮 **חישוב מפורט** - האם מגיע החזר או יש לשלם עוד

## 🚀 שימוש

1. פתח את האתר
2. הזן את ההכנסות וההוצאות שלך
3. המערכת תחשב אוטומטית את המס
